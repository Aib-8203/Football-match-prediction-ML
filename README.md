# Football-match-prediction-ML

## Project Overview
This project uses machine learning to predict whether the home or away team will win a football match using historical match data.

## Dataset
The dataset contains 9,380 matches with 22 features.

## Features Used
- Goal difference
- Team form (last 5 games)
- Average goals conceded
- Shots per game

## Models Implemented
- KNN
- Random Forest
- Logistic Regression

## Results
| Model | Accuracy |
|------|------|
| KNN | 60% |
| Random Forest | 60.8% |
| Logistic Regression | 62.7% |

Logistic Regression performed best on the test set.
